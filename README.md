# Pedagogy Tools

Open educational tools for clinical herbalism pedagogy.

Built with [Claude](https://claude.ai). Hosted on [GitHub Pages](https://johncourie.github.io/pedagogy-tools). No external dependencies beyond Google Fonts.

## Tools

- **When an Herb Gets Labeled Dangerous** — An interactive framework for interrogating herb safety claims, covering causality methods, bias taxonomy, and parallel case studies in green tea and kratom. → [Open](https://johncourie.github.io/pedagogy-tools/herb-safety-tool/)

## Development

All tools are static HTML/CSS/JS — no build step required. Serve locally with any static file server:

```bash
npx serve .
```
